Base de tratamiento

El tratamiento se realiza con base en la autorización del usuario, la gestión de una solicitud precontractual o contractual, el cumplimiento de obligaciones legales y el interés legítimo de mejorar la operación del sitio, según aplique.

Los mensajes de reserva pueden gestionarse mediante WhatsApp. La medición del sitio puede realizarse con herramientas de Google Analytics o Google Tag Manager cuando el usuario lo autorice. Cada tercero procesa información bajo sus propias políticas.

Derechos del titular

El usuario puede solicitar acceso, actualización, corrección, supresión o revocatoria de autorización sobre sus datos personales. Para ejercer estos derechos debe contactar al responsable por los canales oficiales publicados en este sitio.

Conservación

Los datos se conservarán durante el tiempo necesario para atender la finalidad informada, gestionar reservas, cumplir obligaciones legales o resolver solicitudes del usuario.

Seguridad

Aplicamos medidas razonables para proteger la información, incluyendo navegación servida con políticas de seguridad, limitación de rutas públicas y reducción de permisos sensibles del navegador.
